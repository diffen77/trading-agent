# AI Review Rules

Dessa regler granskas automatiskt vid varje push/PR.

## Kodkvalitet
- Ingen duplicerad logik — extrahera till funktioner
- Felhantering på alla async-anrop (try/catch eller .catch)
- Inga hårdkodade secrets, API-nycklar eller lösenord
- Console.log bara i development, aldrig i produktion

## Design & UX
- Responsivt — funkar på 375px (mobil) och 1440px (desktop)
- Tillgänglighet — semantic HTML, alt-text, keyboard navigation
- Kontrast minst 4.5:1 på all text
- Inga generiska CTA:er ("Klicka här") — specifika ("Starta gratis trial")

## Prestanda
- Inga N+1 queries mot databas
- Bilder optimerade (WebP, lazy loading)
- Bundle size — inga onödiga dependencies

## Säkerhet
- Input validering på alla endpoints
- SQL-injection skydd (parameteriserade queries)
- CORS korrekt konfigurerat
- Auth-check på alla skyddade routes

## Konventioner
- TypeScript strict mode
- Svenska i UI-text, engelska i kod
- Commits: conventional commits (feat:, fix:, chore:)
